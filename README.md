# UART_CLI

UART CLI for Elegoo Mega2560 

To build use make, then to flash it use make flash,open up the cli by make cli and dump the elf via make objdump

Commands:
  uptime(counts up to 256 ticks) with an 8-bit timer
  turn LED on and off
